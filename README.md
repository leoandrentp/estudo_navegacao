# Estudos de Navegação Robótica em Grid 2D

Repositório criado para documentar minha evolução nos estudos de lógica de navegação, transição de estados e prevenção de colisão usando Python puro.

## Estrutura do Projeto

* **p1.py**: Criação do espaço de trabalho (matriz 5x5) e intertravamento de limites do mapa usando funções de movimentação manuais.
* **p2.py**: Implementação de lógica "autônoma" baseada em cálculo de erro entre a posição atual e o setpoint (alvo).
* **p3.py**: Adição de obstáculos e algoritmo de checagem preditiva de colisão.

## Como executar

Basta rodar os scripts diretamente no terminal:

python p1.py
python p2.py
python p3.py
